导入struct模块
```python
import struct
```

将不固定长度的数字 ---> 固定的bytes
```python
ret = struct.pack('i', 151988)
print(ret, type(ret), len(ret), sep='\n')

# 返回的结果：
# b'\xb4Q\x02\x00'
# <class 'bytes'>
# 4
```

固定的bytes --->  转为不固定长度的数字
```python
ret = struct.pack('i', 151988)
ret1 = struct.unpack('i', ret)[0]  # 是一个元组格式 (151988,)
print(ret1)  # 151988
```

注意：
```python
# 当转换的数据量大的时候，会报错
```
